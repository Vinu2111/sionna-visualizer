Quick Start 
import requests

API_KEY = "your-api-key-here"
BASE_URL = "https://sionna-visualizer-production-50ae.up.railway.app"

# Run a BER vs SNR simulation
response = requests.post(
    f"{BASE_URL}/v1/api/simulate/ber-snr",
    headers={"X-API-Key": API_KEY},
    json={
        "modulation_order": 4,
        "code_rate": 0.5,
        "snr_min": -5.0,
        "snr_max": 20.0,
        "snr_steps": 25
    }
)

result = response.json()
print(f"Modulation: {result['modulation']}")
print(f"Simulation time: {result['simulation_time_ms']}ms")

# Plot the results
import matplotlib.pyplot as plt
plt.semilogy(result['snr_db'], result['ber_theoretical'],
             label='Theoretical')
plt.semilogy(result['snr_db'], result['ber_simulated'],
             label='Simulated')
plt.xlabel('SNR (dB)')
plt.ylabel('BER')
plt.legend()
plt.show()

Endpoints Reference

POST
/v1/api/simulate/ber-snr

Run an AWGN BER-vs-SNR simulation retrieving theoretical and simulated paths.

ParameterTypeDescription
modulation_orderInteger2 (BPSK), 4 (QPSK), 16 (16QAM), 64 (64QAM)
code_rateFloatLDPC code rate. E.g. 0.5
snr_minFloatStarting SNR in dB
snr_maxFloatEnding SNR in dB
snr_stepsIntegerNumber of points to plot
POST
/v1/api/simulate/beam-pattern

Generates a ULA beam pattern for a given configuration.

ParameterTypeDescription
num_antennasIntegerNumber of elements (e.g. 16)
steering_angleFloatAngle in degrees to steer
frequency_ghzFloatOperating frequency in GHz
array_spacingFloatElement spacing in wavelengths
POST
/v1/api/simulate/modulation-comparison

Compare theoretical BER across all standard modulations.

ParameterTypeDescription
snr_minFloatStarting SNR in dB
snr_maxFloatEnding SNR in dB
snr_stepsIntegerNumber of points to plot
POST
/v1/api/simulate/channel-capacity

Compute Shannon channel capacity across multiple bandwidths based on SNR.

ParameterTypeDescription
snr_minFloatStarting SNR in dB
snr_maxFloatEnding SNR in dB
snr_stepsIntegerNumber of points to evaluate
bandwidths_mhzList[Float]Array of bandwidths to evaluate (e.g. [10.0, 100.0, 400.0])

Python Example:

import requests
response = requests.post(
    "https://sionna-visualizer-production-50ae.up.railway.app/v1/api/simulate/channel-capacity",
    headers={"X-API-Key": "YOUR_API_KEY"},
    json={
        "snr_min": -10.0,
        "snr_max": 30.0,
        "snr_steps": 50,
        "bandwidths_mhz": [10.0, 100.0, 400.0, 1000.0]
    }
)
print(response.json()['insights'])
Authentication

All API endpoints require an API key to be passed in the headers via X-API-Key.

  • Generate an API key by registering and visiting the Dashboard -> API Keys tab.
  • Rate Limits: 100 requests per day per API Key.
curl -X POST \
  https://sionna-visualizer-production-50ae.up.railway.app/v1/api/simulate/ber-snr \
  -H "X-API-Key: YOUR_API_KEY_HERE" \
  -H "Content-Type: application/json" \
  -d '{"modulation_order": 4, "code_rate": 0.5, "snr_min": -5.0, "snr_max": 20.0, "snr_steps": 25}'
